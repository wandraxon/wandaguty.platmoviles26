# wandaguty.platmoviles26

Proyectos de React Native / Expo para Plataformas Moviles.

## Proyectos

- `MiApp`
- `AppNavegacion`
- `MiPrimeraApp`
- `AppApi`

## Consignas cubiertas

- Clase 01 y 01 V2: primera app, texto con nombre, saludo y perfil.
- Clase 02: componentes, props, children, imagenes y botones.
- Clase 03: StyleSheet, Flexbox y estilos condicionales con useState.
- Clase 04: formulario con nombre y contrasena.
- Clase 05: API, ActivityIndicator y FlatList.
- Clase 06: navegacion Stack, parametros y pantalla Acerca de.
